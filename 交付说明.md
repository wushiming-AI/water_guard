## 用户下一步建议

### 1. 初始化数据库

```sql
-- 方式一：执行 init.sql（推荐，全量建库）
mysql -u root -p < C:\Users\AA\Desktop\water_guard\init.sql

-- 方式二：仅建库，让后端自动建表
CREATE DATABASE IF NOT EXISTS drowning_alarm DEFAULT CHARACTER SET utf8mb4;
```

### 2. 安装后端依赖

```bash
pip install fastapi uvicorn aiomysql python-multipart websockets
```

### 3. 启动后端

```bash
cd C:\Users\AA\Desktop\water_guard
python -m uvicorn backend:app --host 0.0.0.0 --port 8000 --reload
```

### 4. 启动检测脚本（可选）

```bash
pip install ultralytics opencv-python requests
python drowning_yolo.py
# 默认使用摄像头，改视频文件只需修改 source 变量
```

### 5. 使用微信小程序

1. 打开微信开发者工具
2. 导入项目：选择 `C:\Users\AA\Desktop\water_guard\miniprogram\`
3. 填入你的 AppID（无真机测试可勾选「不使用云服务」）
4. 在「设置」页面将服务地址改为你本机的局域网 IP（如 `http://192.168.x.x:8000`）  
   ⚠️ 小程序不能访问 localhost，真机调试需改为局域网 IP 或内网穿透地址

### 6. 前端访问

直接用浏览器打开 `index.html` 即可，或通过 Live Server 启动。

## 已知注意事项

- 微信小程序调试时需在开发者工具「详情 → 本地设置」中勾选「不校验合法域名」（开发阶段）
- 正式上线需将后端部署到有公网 IP 的服务器，并配置 HTTPS
- `drowning_yolo.py` 依赖 `best.pt` 模型文件，请确保该文件存在于同目录
